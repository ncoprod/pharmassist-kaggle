# Contracts (Schemas)

This folder will host the **canonical JSON Schemas** shared across the demo:

- `Trace` (agentic steps + tool calls)
- `Run` (pipeline state)
- `PatientContext` (synthetic only for Kaggle)
- `EvidenceItem` (citations + provenance)
- `Product` / `Recommendation`

Day 2 will define the first strict schemas and validators.
